Anmeldung mit deinem GitHub-Token.
Mit „Token merken" bleibst du auf diesem Gerät angemeldet — beim nächsten Öffnen musst du ihn nicht erneut eingeben.
Wie erstelle ich einen Token?
GitHub → Profil → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token.
Bei Repository access nur Website-SR wählen, bei Permissions → Contents auf Read and write setzen. Token kopieren und oben einfügen.
